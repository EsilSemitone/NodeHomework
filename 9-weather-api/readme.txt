Порт 80
GET
http://localhost:80/weather -> Вернет json со списком городов и описанием погоды и температурой,
города и токен будет взят из файла json если параметры будут заданы
GET
http://localhost:80/weather?city={moscow}&token={token} -> Вернет json с выбранным городом и описанием погоды и температурой

POST
http://localhost:80/weather?city={moscow}&token={token} -> Вернет json со статусом "Успешно" в случае успеха записи данных в json на сервере
данные лучше устанавливать по отдельности, не могу понять  почему файл JSON "ломается" при записи нескольких значений

DELETE
http://localhost:80/weather -> Вернет json со статусом "Успешно" в случае успеха удаления фала с конфигурациями (Город, Токен)
